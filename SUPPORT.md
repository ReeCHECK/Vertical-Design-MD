# Support

- Use [GitHub Discussions](https://github.com/ReeCHECK/Vertical-Design-MD/discussions) for questions, pack ideas, and implementation patterns.
- Use [Issues](https://github.com/ReeCHECK/Vertical-Design-MD/issues) for reproducible defects, incorrect guidance, and specification problems.
