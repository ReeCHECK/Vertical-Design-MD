# Enterprise Operations Console

Use this pack for high-volume work queues, governed bulk actions, approvals, and service-level monitoring. It is suitable when agents need to keep tenant scope, selection scope, ownership, priority, SLA, evidence, and partial outcomes visible while acting.

Start with `manifest.json`, then read every required document. Adapt generic object and role names to the product domain before generating a product interface. Backend services remain responsible for authorization, concurrency, tenant isolation, audit, and policy decisions. Security, privacy, accessibility, legal, and domain review remain required before production use.
