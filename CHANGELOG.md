# Changelog

All notable changes to Vertical Design MD are documented here. The project follows semantic versioning for the specification, tools, and reference packs.

## [0.1.0] - 2026-07-22

### Added

- Version 1.0 Vertical Design Pack specification and manifest schema.
- Zero-dependency validation and static catalog generation.
- Fintech trading, healthcare workstation, and enterprise operations reference packs.
- Scenario-based evaluation format and copyable authoring template.
- English and Simplified Chinese project documentation.
- GitHub Actions quality checks and Pages deployment.
- Contribution guide, security policy, support policy, code of conduct, citation metadata, and structured issue/PR templates.

### Known limitations

- No CSS, Tailwind, or Figma token adapters yet.
- No screenshot-based visual regression tests.
- Reference packs have not received a formal domain-expert certification badge.
